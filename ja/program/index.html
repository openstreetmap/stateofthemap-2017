日程概要

State of the Map 2017 は2017年8月18日から20日にかけて福島県会津若松市で開催されます。会津若松市文化センターでお会いしましょう。

  • 8月18日 金曜日: State of the Map の初日です。20分セッションやライトニングトークが予定されています。詳細はタイムテーブルをご覧ください。
  • 8月19日 土曜日: State of the Map 二日目は20分セッションに加えて OSM アワードの発表が予定されています。予定されているセッションはタイムテーブルでご確認ください。
  • 8月20日 日曜日: State of the Map 最終日はマッピングやドキュメント作成などのワークショップが予定されています。一緒に活動することで新しい仲間や知識を発見しましょう。応募の多かったライトニングトークはこの日にも予定されています。

タイムテーブル

素晴らしいセッションとワークショップが多数ラインナップされています。あなたの聞きたくなるセッションが必ずあるはずです。セッションとワークショップの詳細な情報をご覧ください。
